\doxysection{Host side crypto methods (atcah\+\_\+)}
\hypertarget{group__atcah}{}\label{group__atcah}\index{Host side crypto methods (atcah\_)@{Host side crypto methods (atcah\_)}}


Use these functions if your system does not use an ATCADevice as a host but implements the host in firmware. The functions provide host-\/side cryptographic functionality for an ATECC client device. They are intended to accompany the Crypto\+Auth\+Lib functions. They can be called directly from an application, or integrated into an API.
