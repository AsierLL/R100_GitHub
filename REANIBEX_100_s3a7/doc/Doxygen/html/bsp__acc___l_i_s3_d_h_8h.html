◆ Accel_Interrupt_Init()

ERROR_ID_e Accel_Interrupt_Init ( uint8_t enable_axis,
uint8_t threshold,
uint8_t duration,
uint8_t polarity )

Initialize the interruption reg and configure interruption threshold of accelerometer.

Parameters
enable_axisVariable to enable/disable acc axis interruption
threshold7bit, trigger threshold for interruption. Value changes depending on selected scale 1 LSb = 16 mg @ FS = ±2 g 1 LSb = 32 mg @ FS = ±4 g 1 LSb = 62 mg @ FS = ±8 g 1 LSb = 186 mg @ FS = ±16 g
duration7bit, duration of the interruption after triggered, depends of ODR configuration. (t = N/ODR) (e.g: ODR = 25 Hz -> t = 1/25=0,04sec=40msec)
polarityConfigure polarity of interrupt
Returns
acc interrupt errors
